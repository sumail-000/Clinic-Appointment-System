## PowerPoint Design Tips

### Color Scheme
- **Primary:** Professional Blue (#4a90e2)
- **Secondary:** Green (#27ae60) for positive metrics
- **Accent:** Red/Orange (#e74c3c) for problems
- **Background:** White or light grey (#f5f5f5)
- **Text:** Dark grey (#333333)

### Fonts
- **Headings:** Arial Bold, 32-36pt
- **Subheadings:** Arial Bold, 24-28pt
- **Body text:** Arial Regular, 18-20pt
- **Captions:** Arial Regular, 14-16pt

### Icons & Images
- Use simple, professional icons (medical, phone, clock, checkmark, X)
- Use consistent icon style throughout
- Use charts/graphs for quantitative data
- Use arrows for before/after comparisons
- Use color coding (red=problem, green=solution)

### Layout Guidelines
- Keep slides clean and uncluttered
- Use white space effectively
- Limit text to key points (not full sentences)
- Use bullet points with icons
- Use tables for data comparison
- Use visual hierarchy (larger text for important points)

### Slide Transitions
- Use simple, professional transitions (fade or none)
- Avoid flashy animations
- Keep presentation focused on content

## Presentation Delivery Tips

### Timing (15-20 minutes total)
- Context: 4-5 minutes (Slides 1-4)
- Problem: 4-5 minutes (Slides 5-7)
- Solution: 3-4 minutes (Slide 8)
- Features: 5-6 minutes (Slides 9-12)
- Q&A: 5 minutes

### Key Messages to Emphasize
1. **The problem is real and costly:** $19,200 annually wasted
2. **The solution is targeted:** Addresses specific pain points
3. **The features are justified:** Each solves a concrete problem
4. **The impact is measurable:** 50% time reduction, 85% error reduction
5. **The ROI is clear:** System pays for itself in 3 months

### Practice Points
- Know your numbers (call times, error rates, savings)
- Be ready to explain each custom feature's business value
- Connect features back to Canvas problem
- Emphasize alignment between problem and solution
- Show enthusiasm for solving real business problems
